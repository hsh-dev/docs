![Rule Audience First Pn](/images/rule_audience_first.png)

룰 세그먼트를 통해서 직접 조건에 해당하는 유저를 뽑을 수 있어요.

예를 들어 특정 상품을 구매한 유저, 특정 쿠폰을 보유한 유저와 같은 조건을 직접 설정해야 할 때 사용해보세요.

<img
  src="/images/rule_audience_second.png"
  alt="Rule Audience Second Pn"
  title="Rule Audience Second Pn"
  className="mx-auto"
  style={{ width:"58%" }}
/>

필터로는 **이벤트 속성, 유저 속성, 캠페인 수신, 결제 금액**을 기준으로 조건을 추가할 수 있어요.

- 이벤트 속성 : 회원가입, 장바구니, 구매 등 행동 데이터를 기반으로 조건을 설정할 때 사용해보세요.
- 유저 속성 : 유저의 나이, 보유하고 있는 포인트 등 유저 데이터를 기반으로 조건을 설정할 때 사용해보세요.
- 캠페인 수신 : 특정 캠페인을 수신 했거나 수신하지 않은 경우로 조건을 설정할 때 사용해보세요.
- 결제 금액 : 유저의 결제 금액 (order로 계산)을 기준으로 조건을 설정할 때 사용해보세요.

![Rule Audience Third Pn](/images/rule_audience_third.png)

필터 조건을 통해서 설정한 유저의 모수를 확인해보세요.

예상 모수 확인하기를 누르고 기다리면 현재 상태에서 조건에 만족하는 유저수를 정확하게 계산해요.

유저수가 많으면 정확한 숫자를 계산하는데 시간이 많이 걸릴 수 있어요.
