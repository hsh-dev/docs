## **Blux CRM Console 계정 발급 안내**

Blux CRM Console을 사용하려면 데이터 연동이 완료된 후 계정 발급이 필요합니다.

**계정 발급 절차**

1. 데이터 연동 완료 후, 담당자에게 계정 발급을 요청합니다.
2. 발급을 희망하는 고객님의 **성함**과 **전화번호, 이메일을** 담당자에게 전달합니다.
3. 담당자가 안내 메일을 통해 로그인 정보를 제공합니다.

계정 발급과 관련하여 추가 문의 사항이 있을 경우 담당자([elena@blux.ai](mailto:elena@blux.ai))에게 연락해 주세요.

## 발송 채널 등록하기

Blux Console에서 메시지를 발송하려면 캠페인 **발송을 희망하는 채널**별 발신프로필을 등록해야 합니다.

<Note>
  발송 계획이 없는 채널은 등록하지 않으셔도 됩니다.
</Note>

<Steps>
  <Step title="카카오 채널 등록" icon="message" iconType="regular" titleSize="h3">
    카카오 메시지 발송을 위해 **카카오 비즈니스 채널**을 생성하고, 아래 절차를 진행해야 합니다.

    **카카오 채널 등록 절차**

    1. **카카오 비즈니스 채널 생성**
    2. **채널 등록을 위한 토큰 요청**
       - 비즈니스 채널에 대한 아래 정보 전달
       - 플러스친구 ID
       - 관리자 연락처
       - 사업자 카테고리(대분류, 중분류, 소분류)
    3. **관리자 연락처로 받은 6자리 토큰 입력**

    <Check>
      등록 완료 후, 콘솔에서 등록한 채널을 통해 카카오 친구톡 / 알림톡 발송이 가능합니다.
    </Check>
  </Step>
  <Step title="문자메시지 발신번호 등록" icon="message-sms" iconType="regular" titleSize="h3">
    문자 메시지를 통한 캠페인 발송을 위해 발신번호를 등록하고 심사를 요청해야 합니다.

    **발신번호 등록 절차**

    1. **필수 서류 준비 밑 제출**
       - 통신서비스 이용증명원
       - 이용승낙서
       - 사업자등록증
       - 관계 확인 문서 (필요 시)
    2. **등록 및 심사 요청**
    3. **수신거부번호 등록 (Blux에서 진행)**

    <Check>
      심사 완료 후, 등록한 발신번호를 사용하여 문자 메시지를 발송할 수 있습니다.
    </Check>
  </Step>
  <Step title="이메일 발신도메인 등록" icon="envelope" iconType="regular" titleSize="h3">
    이메일 발송을 위해 **발신도메인 등록**이 필요합니다.
    **발신 도메인 등록 절차**

    1. 블럭스 **담당자에게 원하는 도메인 정보 공유 (ex\_ @**[**blux.ai**](http://blux.ai)**)**
    2. 블럭스 **담당자에게 전달받은 TXT 레코드 값** 도메인 DNS에 추가

    <Check>
      등록 완료 후, 등록한 도메인을 사용하여 이메일 발송이 가능합니다.
    </Check>
  </Step>
</Steps>
